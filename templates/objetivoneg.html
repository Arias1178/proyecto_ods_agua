Proyecto ODS: Agua limpia y saneamiento

Objetivos de negocio

Con el modelo de machine learning se busca generar un sistema fiable para saber si el agua, dependiendo de distintos datos, diga si esta contaminada o no y si esta puede llegar a ser tratada. Con el resultado del estado, se puede reducir el tiempo y los costos de los análisis tradicionales, automatizándolos con una alta precisión. Este rápido análisis permite tomar acciones con mayor prontitud para tratar o proteger las fuentes hídricas, apoyando y respaldando las decisiones de entes ambientales encargados.

Objetivo General

Desarrollar una aplicación web interactiva basada en el ODS N.º 6: Agua limpia y saneamiento, que permita predecir la potabilidad del agua mediante técnicas de machine learning, con el fin de concientizar a la población sobre la calidad del agua y promover prácticas responsables de consumo.

Objetivos Específicos

Recolectar y analizar un conjunto de datos sobre la calidad del agua, considerando variables como pH, dureza, sólidos, cloraminas, sulfatos, entre otros.

Entrenar un modelo predictivo (XGBoost) que clasifique si el agua es potable o no, con base en las variables analizadas.

Diseñar e implementar una interfaz web accesible y atractiva que permita al usuario ingresar datos de calidad del agua y obtener un diagnóstico inmediato.

Integrar una campaña educativa dentro de la plataforma, utilizando elementos visuales (como Pokémon de agua) para promover el conocimiento del ODS 6 de forma lúdica e inclusiva.

Publicar el proyecto en un repositorio colaborativo (GitHub o similar) que permita el seguimiento, retroalimentación y futuras mejoras del sistema.
