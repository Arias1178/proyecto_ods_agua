Preguntas Clave

1. ¿Es el agua evaluada segura para el consumo humano? Esta es la pregunta central del modelo. A partir de diversas características físico-químicas del agua, el modelo determina si cumple con las condiciones necesarias para ser considerada potable.

2. ¿Qué variables influyen más en la potabilidad del agua? A través del análisis de los datos y la importancia de las variables en el modelo XGBoost, se busca identificar cuáles factores (como el pH, las cloraminas o los trihalometanos) tienen mayor impacto en la predicción.

3. ¿Se puede automatizar la evaluación de calidad del agua utilizando machine learning? El proyecto intenta demostrar que es posible aplicar modelos supervisados para clasificar la calidad del agua de manera rápida y eficiente.

4. ¿Qué tan confiable es el modelo al predecir la potabilidad del agua en nuevos datos? Se analiza la precisión del modelo para saber si es aplicable en contextos reales y qué margen de error tiene.

5. ¿Cómo puede esta tecnología contribuir a mejorar el acceso a agua limpia en comunidades vulnerables? Se explora el impacto social del uso de modelos predictivos como herramienta de diagnóstico para mejorar la toma de decisiones sobre el agua.
